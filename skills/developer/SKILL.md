# Senior Developer Skills

## Core Competencies
- Full-stack web development (React, Next.js, FastAPI, Node.js, Python)
- Database design (PostgreSQL, SQLite, MongoDB)
- REST API and GraphQL design
- Authentication and authorization (OAuth, JWT)
- Payment integration (Stripe, LemonSqueezy)

## SaaS MVP Playbook
1. **Architecture First**: Design the data model and API before writing UI code
2. **Start with Auth**: Set up user authentication early
3. **Core Feature Only**: Build the one feature that solves the main pain point
4. **Landing Page**: Create a simple landing page with a clear value proposition
5. **Deploy Early**: Get something live on day 1, iterate from there

## Tech Stack Preferences
- **Frontend**: Next.js + Tailwind CSS + shadcn/ui for rapid prototyping
- **Backend**: FastAPI (Python) or Express (Node.js)
- **Database**: SQLite for MVP, PostgreSQL for scale
- **Auth**: NextAuth.js or Supabase Auth
- **Payments**: Stripe Checkout for simplicity
- **Hosting**: Vercel (frontend), Railway/Fly.io (backend)

## Code Standards
- Write clean, typed code (TypeScript / Python type hints)
- Include error handling and input validation
- Write modular, reusable components
- Add setup instructions and environment variable documentation
- Focus on shipping working code fast over perfect architecture
